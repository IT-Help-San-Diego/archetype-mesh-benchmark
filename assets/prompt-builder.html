How This Works

This mapper uses lightweight heuristics — not a workflow engine. When you type a prompt, keywords and tool-call shapes light up the brain regions that the benchmark axes would likely engage for that prompt. Reasoning fires on logic/math/analysis words. Vision fires on images/screenshots. Tools fires on code/execution/API verbs. Literary fires on prose/writing asks. Security fires when the prompt asks the model to override rules or generate risky content. None of this is a guarantee — it’s a teaching surface, not an oracle.
